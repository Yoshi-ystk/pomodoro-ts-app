# 画面フロー

## 起動画面
- `App.tsx` から `PomodoroScreen` を表示
- `PomodoroSettingsProvider` で設定を管理

## Pomodoro Screen（ホーム）
- タイマー表示（mm:ss形式、Orbitronフォント）
- 現在のフェーズ表示（作業/短休憩/長休憩）
- 周回数表示（○周目）
- 開始/一時停止ボタン
- リセットボタン
- 右上に設定ボタン

遷移先:
- Settings Screen

## Settings Screen
- フェーズ表示名のカスタマイズ
  - 作業フェーズの表示名
  - 短休憩フェーズの表示名
  - 長休憩フェーズの表示名
- カスタムモードの切り替えスイッチ
- カスタムモード時の設定項目（カスタムモードOFF時は無効化）
  - 作業時間（15〜60分）
  - 短休憩時間（3〜10分）
  - 長休憩時間（10〜30分）
  - 1セットあたりの作業回数（3〜6回）
- 保存ボタン
- 戻るボタン

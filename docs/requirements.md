# 要件定義（FOCUSTEP - Pomodoro Timer App）

## 目的
集中力維持と習慣化のため、ポモドーロ・テクニックをスマートフォンで実行できるアプリを提供する。

## 対象
・個人利用を想定
・将来的には一般公開も可能な設計

## 主な機能
1. ポモドーロタイマー
   - 作業フェーズ
   - 短休憩フェーズ
   - 長休憩フェーズ
   - 1セット = 作業4回（デフォルト）+ 長休憩 のサイクル管理
   - 自動フェーズ遷移
   - 周回数の表示

2. 設定機能
   - **基本モード**: デフォルト設定（25分作業、5分短休憩、15分長休憩、4回で1セット）
   - **カスタムモード**: ユーザーが自由に設定可能
     - 作業時間：15〜60分
     - 短休憩時間：3〜10分
     - 長休憩時間：10〜30分
     - 1セットあたりの作業回数：3〜6回
   - フェーズ表示名のカスタマイズ
   - 設定値はアプリ再起動後も保持（AsyncStorage）

3. UI
   - タイマー表示（mm:ss形式、Orbitronフォント使用）
   - 開始／一時停止
   - リセット
   - 設定画面（右上の設定ボタンから遷移）
   - ナンバーピッカー（長押し機能、直接入力対応）

4. 将来機能
   - タスク名の入力
   - フォーカス・ログの記録
   - 日/週単位の統計表示
   - 効果音通知
   - 通知機能

## 非機能要件
・React Native + Expo
・iOS / Android 両対応
・ローカルデータ保存：AsyncStorage
・軽量・ミニマルな UI

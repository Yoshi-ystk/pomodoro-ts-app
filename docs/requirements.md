# 要件定義（FOCUSTEP - Pomodoro Timer App）

## 目的
集中力維持と習慣化のため、ポモドーロ・テクニックをスマートフォンで実行できるアプリを提供する。

## 対象
・個人利用を想定
・将来的には一般公開も可能な設計

## 主な機能
1. ポモドーロタイマー
   - 作業フェーズ
   - 短休憩フェーズ
   - 長休憩フェーズ
   - 1セット = 作業4回 + 長休憩 のサイクル管理

2. 設定機能
   - 長休憩時間
   - 1セットあたりの作業回数
   - 設定値はアプリ再起動後も保持（AsyncStorage）

3. UI
   - タイマー表示（mm:ss）
   - 開始／停止
   - リセット
   - 設定画面（ハンバーガーメニュー or アイコン遷移）

4. 将来機能
   - タスク名の入力
   - フォーカス・ログの記録
   - 日/週単位の統計表示
   - 効果音通知

## 非機能要件
・React Native + Expo
・iOS / Android 両対応
・ローカルデータ保存：AsyncStorage
・軽量・ミニマルな UI
